## 測試方法

### 方法 1: 使用測試頁面（最簡單）⭐

1. **打開測試頁面**
   - 雙擊 `frontend\test.html`
   - 或在瀏覽器中打開該文件

2. **測試步驟**
   - 點擊「選擇文件」或拖放圖片
   - 選擇一張文檔圖片（PNG, JPG, JPEG）
   - 點擊「識別文檔」
   - 等待處理完成
   - 查看識別結果

3. **預期結果**
   - 顯示文檔類型
   - 顯示置信度
   - 顯示提取的信息（如果有）
   - 顯示OCR識別的文字

1. **運行測試腳本**

2. **使用自己的圖片測試**

### 方法 3: 使用瀏覽器直接測試API

1. **測試後端健康狀態**
   - 訪問: http://localhost:5000
   - 應該看到 JSON 響應

2. **使用瀏覽器開發者工具**
   - 按 F12 打開開發者工具
   - 在 Console 中測試 API

## 常見問題

### Q: 上傳失敗？
**A**: 檢查：
- 後端是否運行
- 文件大小是否超過 10MB
- 文件格式是否支持（PNG, JPG, JPEG）

### Q: OCR 顯示模擬結果？
**A**: 這是正常的，因為 PaddleOCR 未安裝。
- 安裝 PaddleOCR: `pip install paddleocr`
- 或繼續使用模擬模式測試界面

### Q: 無法連接後端？
**A**: 檢查：
- 後端是否在運行
- 端口 5000 是否被占用
- 防火牆是否阻止連接
