## Topic:

本應用是一個智能文檔識別與信息提取系統，能夠識別香港常見文檔類型（身份證、水電費單、銀行賬單等），並自動提取關鍵信息（地址、姓名、日期等）。系統提供Web應用界面，支持文檔上傳、識別、信息提取、地圖顯示和隱私保護功能。

---

## Machine Learning Task:

本應用使用了**圖像分類（Image Classification）**和**光學字符識別（OCR）**兩種機器學習任務。

### 圖像分類任務
- **任務類型**: 多類別圖像分類（Multi-class Image Classification）
- **選擇原因**: 
  - 需要識別6種不同的文檔類型
  - 圖像分類是深度學習中最成熟的任務之一
  - 可以使用遷移學習快速訓練高準確率模型

### OCR任務
- **任務類型**: 光學字符識別（Optical Character Recognition）
- **選擇原因**:
  - 需要從文檔圖片中提取文字信息
  - PaddleOCR支持中英文識別，適合香港場景
  - 可以識別繁體中文和英文混合文檔
